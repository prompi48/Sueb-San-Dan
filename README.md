# โปรเจกต์สืบสันดาน (SeubSunDan)
แพลตฟอร์มส่งต่อมรดกความรู้จากรุ่นพี่สู่รุ่นน้องอย่างเป็นระบบ

## เกี่ยวกับโปรเจกต์
โปรเจกต์นี้สร้างขึ้นเพื่อเป็นศูนย์กลางการแบ่งปัน "มรดกความรู้" เช่น สรุปบทเรียน เทคนิคการเรียน และข้อสอบเก่า โดยเน้นความถูกต้องผ่านระบบ Admin Approval

## Key Features
- **Authentication:** ระบบยืนยันตัวตนแยก Role (Student / Admin)
- **Legacy CRUD:** สร้าง, อ่าน, แก้ไข, และลบโพสต์มรดกความรู้
- **Admin Moderation:** ระบบตรวจสอบและอนุมัติโพสต์ก่อนเผยแพร่จริง
- **Cloud Deployment:** ใช้งานผ่านระบบคลาวด์เต็มรูปแบบ

## Tech Stack
- **Frontend:** Next.js (App Router)
- **Backend/Database:** Supabase (PostgreSQL)
- **Deployment:** Vercel
